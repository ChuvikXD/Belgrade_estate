# Belgrade Estate Bot

Телеграм-бот для поиска квартир в аренду в Белграде, Сербия.

## Возможности
- Поиск квартир на естественном языке через LLM
- Фильтры: район, улица, цена, площадь, мебель, этаж, питомцы, отопление, кол-во комнат
- Фото объявлений
- Ссылка на карту и полное объявление

## Структура проекта
- `bot.py` - основной файл телеграм-бота (aiogram)
- `llm_prettified.py` - извлечение параметров фильтра из текста через LLM (Groq API)
- `image_parser.py` - получение фотографий объявлений
- `parser.py` - скрапинг данных с cityexpert.rs(для обновления базы данных требуется запускать его вручную, либо автоматизировать работу скрипта на сервере)
- `apartments.csv` - датасет с объявлениями
- `visuals.ipynb` - аналитика рынка недвижимости

## Установка
\`\`\`bash
pip install -r requirements.txt
\`\`\`

Создай файл `.env`:
\`\`\`
TOKEN=твой_telegram_bot_token
GROQ_API_KEY=твой_groq_key
\`\`\`

## Запуск
\`\`\`bash
python bot.py
\`\`\`
